## Project Overview
This project demonstrates a simple AI-powered chatbot built using Python, Streamlit, and the OpenAI API. The application allows users to input a question and receive a generated response from a language model. The goal of the project is to experiment with integrating modern AI APIs into a lightweight software application while following basic Agile development practices.

The chatbot performs a basic question-answering task. Users type a question into the interface and the AI model generates a short response. This mini-lab introduces core ideas behind AI integration, prompt engineering, and responsible AI use.

---

## Model Information

**Model Name:** GPT-4.1-mini  
**Source:** OpenAI API  
**Provider:** OpenAI

---

## Rationale for Model Selection
GPT-4.1-mini was selected because it is a lightweight and efficient language model designed for fast responses in conversational applications. It provides strong natural language understanding while remaining cost-effective for small applications and experiments. This makes it well suited for building simple prototypes such as chatbots or question-answering systems.

The model integrates easily with Python through the OpenAI API, allowing developers to quickly experiment with prompt engineering and AI-powered interactions. For this mini-lab, the model was used to generate short responses to user questions within a Streamlit interface.
